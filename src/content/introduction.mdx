## What is LiteDocs?

LiteDocs is a simple, fast template for building documentation sites with Next.js and Tailwind CSS. Content is written in MDX and compiled by Content Collections.

## Key Features

- MDX with React components
- Dark mode and responsive UI
- Content Collections-driven routing and slug generation
